Les Questions Les Plus Fréquentes

Voici les questions les plus fréquentes qui sont posées avant de contacter une agence web ou agence graphique.

Quels Sont Les Tarifs De La Conception D’un Logo Professionnel En Tunisie?
  • Le prix de la création d’un logo professionnel, élaboré par une bonne agence de communication en Tunisie ayant des graphistes expérimentés se situe généralement entre 400 dt et 3500 dt selon l’envergure et les contraintes du projet.
Quel Est Le Prix D’une Charte Graphique En Tunisie?
  • Conception d’une charte graphique pour une entreprise en phase de démarrage d’activité : à partir de 700 dt.
  • Conception d’une charte graphique pour une TPE (- de 10 personnes) : à partir de 1100 dt.
  • Conception d’une charte graphique pour une PME (entre 11 et 199 personnes) : à partir de 1700 dt.
  • Conception d’une charte graphique pour une grande entreprise (+ de 200 personnes) : à partir de 4500 dt.
  • Conception d’une charte graphique d’une marque de vêtements : à partir de 2600 dt.
  • Conception d’une charte graphique d’une marque de produits cosmétiques : à partir de 3000 dt.
Quel est le prix de développement d’un site web en Tunisie?
  • Le coût minimal pour développer un site onepage professionnel en Tunisie commence à partir de 1800 dt.
  • Le coût minimal pour développer un site vitrine professionnel en Tunisie commence à partir de 3200 dt.
  • Le coût minimal pour développer un site e-commerce professionnel en Tunisie commence à partir de 4000 dt.
